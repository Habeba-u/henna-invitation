# 💍 Henna Invitation – دعوة حنّة نعمة

A luxurious Arabic henna invitation landing page with an interactive experience, elegant design, and personalized guest messages.

---

## ✨ Features

* 🎀 Interactive envelope opening animation
* 🎶 Background music experience
* 💌 Send a personalized message to the bride
* 🎨 Customize message (colors, font, size)
* 📍 Venue location with Google Maps
* ⏳ Countdown to the event
* 💫 Smooth animations & premium UI
* 📩 Messages are stored using Formspree
